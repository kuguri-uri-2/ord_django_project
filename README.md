# ORD — приложение для изучения слов с помощью карточек

## 📌 Назначение проекта

**ORD** — это веб-приложение на базе Django, предназначенное для изучения иностранных слов с помощью карточек.

Приложение позволяет пользователям:
- создавать собственные карточки слов  
- группировать их по темам  
- добавлять перевод и примеры использования  

Такой подход помогает лучше запоминать лексику и видеть слова в контексте.

### Основные цели проекта:
- предоставить удобный инструмент для изучения иностранных слов  
- позволить пользователям создавать и редактировать словарные карточки  
- организовывать слова по категориям (темам)  
- хранить переводы и примеры использования слов  
- продемонстрировать использование фреймворка Django при разработке веб-приложений  

---

## 🚀 Основные возможности

Приложение поддерживает следующие функции:

- просмотр списка всех карточек  
- фильтрация карточек по категории  
- поиск карточек по слову или примеру использования  
- просмотр детальной информации о карточке  
- создание новых карточек слов  
- редактирование существующих карточек  
- удаление карточек (доступно только автору)  
- группировка карточек по темам  
- управление данными через административную панель Django  

### Каждая карточка содержит:
- слово  
- перевод  
- пример использования  
- категорию  
- автора карточки

## ⚙️ Инструкция по запуску проекта

Для запуска проекта выполните следующие шаги:

### 1. Скачать проект

Скачайте проект или клонируйте репозиторий:

```bash
git clone <ссылка_на_репозиторий>
```

Либо скачайте архив проекта и распакуйте его.

---

### 2. Перейти в папку проекта

```bash
cd ord
```

---

### 3. Установить зависимости

Убедитесь, что у вас установлен **Python** и **Django**.

Если Django не установлен:

```bash
pip install django
```

---

### 4. Применить миграции базы данных

```bash
python manage.py makemigrations
python manage.py migrate
```

---

### 5. Создать администратора

```bash
python manage.py createsuperuser
```

Следуйте инструкциям в терминале.

---

### 6. Запустить сервер разработки

```bash
python manage.py runserver
```

---

### 7. Открыть приложение в браузере

**Главная страница приложения:**
```
http://127.0.0.1:8000/
```

**Административная панель Django:**
```
http://127.0.0.1:8000/admin/
```
